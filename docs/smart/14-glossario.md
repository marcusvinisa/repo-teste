# 14 — Glossário

> Termos PT-BR/EN usados na suite Smart. Ordem alfabética.

| Termo | Definição |
|---|---|
| **aFRR** | *Automatic Frequency Restoration Reserve* — reserva de restabelecimento de frequência (serviço ancilar). |
| **ACL** | Ambiente de Contratação Livre — mercado livre de energia. |
| **ACR** | Ambiente de Contratação Regulada — mercado cativo (tarifa da distribuidora). |
| **ANATEL** | Agência Nacional de Telecomunicações — homologa produtos com radiofrequência. |
| **ANEEL** | Agência Nacional de Energia Elétrica — reguladora do setor elétrico. |
| **Anti-ilhamento** | Proteção que desconecta a geração quando a rede cai, evitando energizar a rede ("ilha") — função do inversor. |
| **Autoconsumo (self-consumption)** | Uso da energia gerada na própria UC antes de importar/exportar. |
| **Autoconsumo remoto** | GD em UC distinta, mesma titularidade e concessão, compensando consumo de outra UC. |
| **Bandeiras tarifárias** | Adicional tarifário (verde/amarela/vermelha 1 e 2) conforme custo de geração. |
| **BMS** | *Battery Management System* — gestão da bateria. |
| **CAN** | *Controller Area Network* — barramento usado por BMS/PCS. |
| **CCEE** | Câmara de Comercialização de Energia Elétrica — contabiliza e liquida o mercado. |
| **CSIP** | *Common Smart Inverter Profile* — perfil de controle de DER sobre IEEE 2030.5. |
| **Curtailment** | Limitação/redução comandada de geração (ou carga). |
| **DAL** | *Device Abstraction Layer* — camada que abstrai dispositivos multimarca. |
| **DER** | *Distributed Energy Resources* — recursos energéticos distribuídos. |
| **Demand Response (resposta da demanda)** | Ajuste do consumo em resposta a sinais de preço/rede. |
| **DLMS/COSEM** | Protocolo de leitura de medidores de energia. |
| **EEBus** | Padrão de interoperabilidade para gestão de energia doméstica (ex.: bombas). |
| **EMS** | *Energy Management System* — sistema de gestão de energia (escopo amplo). |
| **EMUC** | Empreendimento com Múltiplas Unidades Consumidoras (ex.: condomínios) na GD. |
| **ESS** | *Energy Storage System* — sistema de armazenamento (bateria). |
| **FCR** | *Frequency Containment Reserve* — reserva de contenção de frequência. |
| **Feed-in** | Energia injetada na rede pela geração. |
| **Fio B** | Componente da TUSD (uso do fio) cobrado gradualmente sobre energia compensada na GD (Lei 14.300). |
| **GD** | Geração Distribuída (micro/minigeração). |
| **Geração compartilhada** | Consórcio/cooperativa que rateia créditos de GD entre UCs. |
| **HEMS** | *Home Energy Management System* — gestão de energia residencial. |
| **IAM** | *Identity and Access Management* — identidade e controle de acesso. |
| **IEC 61850** | Norma de comunicação para automação de subestações/DER. |
| **IEEE 2030.5** | Padrão de comunicação para DER (base do CSIP). |
| **INMETRO** | Instituto que define etiquetagem/certificação compulsória (inversores/módulos). |
| **Load shifting** | Deslocamento de consumo/carga para horários mais baratos. |
| **LGPD** | Lei Geral de Proteção de Dados. |
| **Matter** | Padrão de casa inteligente (com tipos de dispositivos de energia). |
| **Mercado cativo** | Consumidor atendido pela distribuidora na tarifa regulada. |
| **Mercado livre** | Consumidor que contrata energia de comercializador (ACL). |
| **Microgeração** | GD com potência ≤ 75 kW. |
| **Minigeração** | GD acima de 75 kW até o limite regulatório. |
| **Modbus (RTU/TCP)** | Protocolo industrial de leitura/escrita de registradores (serial/Ethernet). |
| **MPPT** | *Maximum Power Point Tracking* — rastreamento de máxima potência (string do inversor). |
| **mTLS** | TLS mútuo — autenticação de ambos os lados do canal. |
| **MQTT / Sparkplug** | Protocolo de mensageria leve / especificação para telemetria industrial sobre MQTT. |
| **OCPP** | *Open Charge Point Protocol* — controle de carregadores de VE. |
| **ONS** | Operador Nacional do Sistema Elétrico. |
| **OTA** | *Over-the-Air* — atualização remota de firmware. |
| **PCS** | *Power Conversion System* — conversor da bateria. |
| **Peak shaving** | Redução de picos de demanda. |
| **PLD** | Preço de Liquidação das Diferenças — preço de curto prazo no mercado. |
| **PRODIST** | Procedimentos de Distribuição da ANEEL (Módulo 3 trata da conexão de GD). |
| **RBAC** | *Role-Based Access Control* — controle de acesso por papel. |
| **REN** | Resolução Normativa da ANEEL (ex.: REN 1.000/2021). |
| **SCEE** | Sistema de Compensação de Energia Elétrica — créditos de GD. |
| **Secure boot** | Inicialização que só aceita firmware assinado. |
| **SG-Ready** | Interface de gestão de energia de bombas de calor (4 estados). |
| **SoC** | *State of Charge* — estado de carga da bateria. (Também *System on Chip* em hardware — ver contexto.) |
| **SoH** | *State of Health* — saúde da bateria. |
| **SunSpec** | Modelos Modbus padronizados para inversores/baterias/medidores. |
| **TE** | Tarifa de Energia. |
| **ToU** | *Time-of-Use* — tarifa por horário (no BR, Tarifa Branca). |
| **TSDB** | *Time-Series Database* — banco de séries temporais. |
| **TUSD** | Tarifa de Uso do Sistema de Distribuição. |
| **UC** | Unidade Consumidora — ponto de conexão faturável. |
| **V1G** | Carga inteligente unidirecional de VE (modulação). |
| **V2G / V2H** | *Vehicle-to-Grid* / *Vehicle-to-Home* — energia do VE para a rede/casa. |
| **VPP** | *Virtual Power Plant* — usina virtual (agregação de DERs). |
| **Zero-export** | Operação que impede injeção de energia na rede. |
