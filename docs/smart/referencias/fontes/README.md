# fontes/ — documentos originais de referência (pesados)

> Onde entram **documentos pesados** fornecidos como referência (ex.: a **matriz de produtos para modelos de negócio**, datasheets, PDFs de estudo) — convertidos para Markdown.

## Como adicionar
1. **Não cole PDFs/planilhas grandes no chat** — pesam demais e estouram o contexto.
2. Converta para Markdown com **[markitdown](https://github.com/microsoft/markitdown)** (PDF, DOCX, XLSX, PPTX → `.md`):
   `markitdown matriz-produtos.xlsx > fontes/matriz-produtos.md`
3. Commite o **`.md`** aqui. Para eu **revisar a lógica × negócios/produtos**, basta o `.md` (ou colar o texto convertido).
