# 14 — Glossary (EN)

> EN/PT-BR terms used across the Smart suite. PT-BR source: [`../14-glossario.md`](../14-glossario.md).

| Term | Definition |
|---|---|
| **aFRR** | Automatic Frequency Restoration Reserve (ancillary service). |
| **ACL** | Ambiente de Contratação Livre — Brazil's free energy market. |
| **ACR** | Ambiente de Contratação Regulada — captive/regulated market. |
| **ANATEL** | Brazil's telecom agency — homologates RF products. |
| **ANEEL** | Brazil's electricity regulator. |
| **Anti-islanding** | Protection that disconnects generation when the grid fails — an inverter function. |
| **Self-consumption** | Using locally generated energy on-site before importing/exporting. |
| **Remote self-consumption** | DG at a different CU (same owner/concession) offsetting another CU. |
| **Tariff flags (bandeiras)** | Surcharge (green/yellow/red 1 & 2) per generation cost. |
| **BMS** | Battery Management System. |
| **CAN** | Controller Area Network — bus used by BMS/PCS. |
| **CCEE** | Brazil's energy clearing chamber. |
| **CSIP** | Common Smart Inverter Profile (DER control over IEEE 2030.5). |
| **Curtailment** | Commanded reduction of generation (or load). |
| **DAL** | Device Abstraction Layer — abstracts multi-brand devices. |
| **DER** | Distributed Energy Resources. |
| **Demand response** | Adjusting consumption in response to price/grid signals. |
| **DG (GD)** | Distributed Generation (micro/mini). |
| **DLMS/COSEM** | Energy-meter reading protocol. |
| **EEBus** | Home energy-management interoperability standard (e.g., heat pumps). |
| **EMS** | Energy Management System (broad scope). |
| **EMUC** | Multiple-CU development (e.g., condominiums) in DG. |
| **ESS** | Energy Storage System (battery). |
| **FCR** | Frequency Containment Reserve. |
| **Feed-in** | Energy injected into the grid by generation. |
| **Fio B** | TUSD wire-use component charged gradually on compensated DG energy (Law 14,300). |
| **HEMS** | Home Energy Management System. |
| **IAM** | Identity and Access Management. |
| **IEEE 2030.5** | DER communication standard (basis for CSIP). |
| **INMETRO** | Brazilian institute defining compulsory certification (inverters/modules). |
| **Load shifting** | Moving consumption/load to cheaper hours. |
| **LGPD** | Brazil's General Data Protection Law. |
| **Matter** | Smart-home standard (with energy device types). |
| **Microgeneration** | DG ≤ 75 kW. **Minigeneration** | DG above 75 kW up to the regulatory limit. |
| **Modbus (RTU/TCP)** | Industrial register read/write protocol (serial/Ethernet). |
| **MPPT** | Maximum Power Point Tracking. |
| **mTLS** | Mutual TLS — both channel ends authenticated. |
| **MQTT / Sparkplug** | Lightweight messaging / industrial telemetry spec over MQTT. |
| **OCPP** | Open Charge Point Protocol — EV charger control. |
| **ONS** | Brazil's national grid operator. |
| **OTA** | Over-the-Air firmware update. |
| **PCS** | Power Conversion System (battery converter). |
| **Peak shaving** | Reducing demand peaks. |
| **PLD** | Short-term settlement price in the Brazilian market. |
| **PRODIST** | ANEEL distribution procedures (Module 3 covers DG connection). |
| **RBAC** | Role-Based Access Control. |
| **REN** | ANEEL Normative Resolution (e.g., REN 1,000/2021). |
| **SCEE** | Electricity Compensation System — DG credits. |
| **Secure boot** | Boot that only accepts signed firmware. |
| **SG-Ready** | Heat-pump energy-management interface (4 states). |
| **SoC** | State of Charge (battery). Also System on Chip (hardware) — by context. |
| **SoH** | State of Health (battery). |
| **SunSpec** | Standardized Modbus models for inverters/batteries/meters. |
| **TE / TUSD** | Energy tariff / Distribution system use tariff. |
| **ToU** | Time-of-Use (in BR, the White Tariff). |
| **UC (CU)** | Consumer Unit — billable connection point. |
| **V1G / V2H / V2G** | Smart unidirectional EV charging / Vehicle-to-Home / Vehicle-to-Grid. |
| **VPP** | Virtual Power Plant (DER aggregation). |
| **Zero-export** | Operation preventing energy injection to the grid. |
